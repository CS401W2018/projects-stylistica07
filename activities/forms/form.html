Engagement Questions

  • What are different ways to style your form? -There are several ways to style a form, including using CSS Flexbox and CSS Grid for responsive layouts that adapt well to different screen sizes, and table layouts for a more rigid, but less modern structure. Elements like fieldsets, legends, placeholders, and labels improve organization and usability, while CSS pseudo-classes and JavaScript can add interactive effects, enhancing user experience. Frameworks like Bootstrap offer pre-built styles and responsive designs, making it easier to create polished forms quickly.

  • Why does the professor like to use flex? -Flex allows for responsive layouts that automatically adjust to fit various screen sizes, making forms more user-friendly. Flexbox also offers powerful alignment and spacing controls, which simplify the layout process and reduce the need for extra code.
