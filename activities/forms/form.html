Engagement Questions

  • What is JavaScript? How did it get its name? -JavaScript is a programming language commonly used to create interactive effects within web browsers. Originally named Mocha, it was later renamed JavaScript in a marketing move to capitalize on the popularity of Java, although the two languages are distinct.

  • What are the two ways you can run JavaScript? -JavaScript can be run in two main ways: in the browser and on the server. In the browser, JavaScript code is executed by the browser's JavaScript engine, enabling interactive and dynamic web pages. On the server, platforms like Node.js allow JavaScript to be used for backend development, expanding its functionality beyond the client side.

  • What was the most popular front-end js library? -The most popular front-end JavaScript library has historically been jQuery. Known for simplifying HTML DOM manipulation, event handling, and cross-browser compatibility, jQuery became widely adopted and set the foundation for modern JavaScript development.
