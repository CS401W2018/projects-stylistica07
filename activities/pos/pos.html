Answers to Engagement Questions

  1. Block tags take up the full width and start on a new line. Inline tags only take up as much space as their content and stay in line with other elements.
  2. Normal flow is how HTML elements naturally arrange themselves on a page. Block elements stack vertically, while inline elements line up next to each other.
  3. Float moves an element to the left or right and allows text or other elements to wrap around it. Clearing the float ensures the parent container adjusts to the floated element's height..
  4. Absolute positioning places an element relative to its nearest positioned ancestor or the page itself. Fixed positioning keeps the element in place even when the page scrolls, while relative positioning shifts the element from its original spot but keeps it in the normal flow.
