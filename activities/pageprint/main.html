  1. We need a viewport meta tag to ensure that a web page scales properly on mobile devices. It controls how a page is displayed on different screen sizes, especially for responsive design.
  2. When two styles with different flex-direction values apply to the same tag, the style that appears last in the CSS file takes priority. CSS applies rules in the order they are written, with the latest one overriding previous styles for the same property.
  3. To change the CSS for a webpage displayed on a phone, use media queries with conditions like @media (max-width: 600px). These queries allow specific styles to apply only when the screen size is within a specified range, adjusting the layout for mobile devices.
