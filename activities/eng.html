What is Structural Tags? (HEADER, NAV, MAIN, FOOTER).

Header - often contains the logo, site title and basic contact inforamtion.
Navigation - Inclueds links to other pages or sections of the website.
Main - this is the central part of the webpage. It contains all the important information the webpage is intended to communicate to the visitor such as text, images and other media.
Footer - this section often contains copyright information, links to privacy policies, terms of use. It aslo contains social media links.
HTML Validation - Ensures that HTML code is correct and adheres to web standards which helps with cross-browser compatibility and SEO.
