Engagement - Beginning CSS

1. Explain the difference between inline, embedded, and external CSS.

  • Inline CSS is used directly within an HTML tag using the 'style' attribute. It affects only that specific element.
  • Embedded CSS (or internal CSS) is placed within the "style" tags in the "head" section of an HTML document, affecting elements on that page based on the specified selectors.
  • External CSS involves linking a separate .css file to the HTML document using the "link" tag.
    • This file can style multiple web pages if they all link to it.

2. Explain why you should only use external CSS.

  • Using external CSS promotes cleaner HTML and makes your website easier to maintain. It allows you to change styles across multiple pages by editing a single file, ensures consistent styling, and can improve page load times when cached by the browser.

3. What is a selector used for?

  • A selector is used to target HTML elements and apply specific styles to them. It identifies which elements the CSS rules will affect, based on elements' attributes like id, class, or type.
