Engagement Questions

  1. What you learned about Flex and CSS?
    I learned that CSS Flexbox is a powerful layout system that makes it easy to align and distribute elements within a container. It allows for flexible and responsive designs by enabling items to automatically adjust their size and position based on available space, without the need for complex calculations.

  2. Why do you need to have an accessible website? What does that mean?
    An accessible website allows everyone, including people with disabilities, to use it easily. This means designing the site so that it works for all users and follows guidelines to make it inclusive for everyone.
