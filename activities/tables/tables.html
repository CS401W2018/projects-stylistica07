Engagement Questions

  1. Why don't we use the border property? - Using the border attribute in HTML is outdated; CSS should be used to handle styling for better separation of content and design. CSS provides more flexibility and control over borders, making it the preferred approach in modern web development.
  2. What is the easiest way to color every other row a different color? - The simplest way to color every other row is by using tr:nth-child(even) in CSS. This applies the specified background color to all even-numbered rows, creating an alternating row effect.
  3. When should we use a table? - Use a table when presenting data that is structured in rows and columns, such as schedules, financial data, or comparison charts. Tables are ideal for content where the relationship between data points is essential for understanding, like tabular data or data that requires headers.
