Difference between an absolute link and a relative link.

An absolute link provides the full web address to a specific page, including the domain name, while a relative link only specifies the location of a page relative to the current page, meaning it doesn't include the full domain name.

List the 3 rules with relative links.

  1. one period . - current folder
  2. two periods .. - parent folder
  3. slash / - start at the root

Do links always need to be inside a NAV tag?

NOT all links of a document should be inside a "NAV" element.
